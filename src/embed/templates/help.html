Embed a Pathway

Copy any of the examples below to embed a WikiPathways pathway into your own site.

Basic

Just embed a pathway:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>

Highlight Nodes

You can highlight by identifier or by node text. Specify a recognized CSS color or hex code followed by the identifier or node text.

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?lightgreen=Entrez_Gene_10400&pink=CAS_62-49-7&lightblue=Entrez_Gene_1119"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>

Mapped Identifiers

Pathway WP528 uses Entrez Gene (NCBIGene) for identifiers, but you can highlight using Ensembl identifiers if you prefer:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528&yellow=Ensembl_ENSG00000110721,Ensembl_ENSG00000161217"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>
