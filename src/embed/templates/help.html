Embed a Pathway

Copy any of the examples below to embed a WikiPathways pathway into your own site.

Basic

Just embed a pathway:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528"
            width="600px"
            height="300px"
            style="overflow:hidden; border:solid;">
          </iframe>

Highlight Nodes

You can highlight by identifier (Entrez_Gene, Ensembl, ChEBI, HMDB or Wikidata) or by node label. Specify a recognized CSS color name or hex value and an identifier or node label like this:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?lightgreen=PEMT&lightpink=Choline"
            width="600px"
            height="300px"
            style="overflow:hidden; border:solid;">
          </iframe>

Equivalent with hex values and node identifiers:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?90ee90=Entrez_Gene_10400&ffb6c1=HMDB_HMDB0000097"
            width="600px"
            height="300px"
            style="overflow:hidden; border:solid;">
          </iframe>

To specify multiple identifiers for one color, separate with commas.

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?lightgreen=Entrez_Gene_10400,Entrez_Gene_1103"
            width="600px"
            height="300px"
            style="overflow:hidden; border:solid;">
          </iframe>

Mapped Identifiers

We map all identifiers to the those in the list above, so you can feel free to use whichever ones you prefer. For example the authors of pathway WP528 specified Entrez Gene (NCBIGene) identifiers, but you can highlight with Ensembl:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?lightgreen=Ensembl_ENSG00000133027,Ensembl_ENSG00000070748"
            width="600px"
            height="300px"
            style="overflow:hidden; border:solid;">
          </iframe>
