Embed a Pathway

Copy any of the examples below to embed a WikiPathways pathway into your own site.

Basic

Just embed a pathway:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>

Highlight Nodes

You can highlight by identifier (Entrez_Gene, Ensembl, ChEBI, HMDB or Wikidata) or by node label. Specify a recognized CSS color name or hex value followed by the identifier or node text. If you want multiple identifiers for one color, separate with commas.

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?lightgreen=Entrez_Gene_10400,Entrez_Gene_1119&ff000080=HMDB_HMDB0000097&0000ff80=CHKA"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>

Mapped Identifiers

We map all identifiers to the those in the list above, so you can feel free to use whichever ones you prefer. For example the authors of pathway WP528 specified Entrez Gene (NCBIGene) identifiers, but you can highlight with Ensembl:

        
          <iframe
            src="https://pathway-viewer.toolforge.org/embed/WP528?yellow=Ensembl_ENSG00000110721,Ensembl_ENSG00000161217"
            width="600px"
            height="300px"
            style="overflow:hidden;">
          </iframe>
